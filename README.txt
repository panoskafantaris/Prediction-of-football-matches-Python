Η βαση δεδομενων λογω ογκου δε  γινοταν να συμπεριληφθει στο repository.
Οποιος θελει ωστοσο να χρησιμοποιησει το κωδικα πρεπει να αντικαταστησει 
το 'database.sqlite' με τη δικια του βαση δεδομενων, στο αρχειο Data_Tables.py
και φυσικα να συμπεριλαβει τα columns που θελει. 

Για καλυτερη κατανοηση του κωδικα δειτε Aναγνωριση_Προτυπων_αναλυση
